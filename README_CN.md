# 🏛️ 一人之国 Nation of One

![Nation of One](assets/cover.jpg)

> [English README](README.md)

---

效率工具把你当成一条流水线。输入任务，输出结果，循环往复。

但人不是流水线。人是一个不断成为自己的过程。

目标从来不是做更多。而是成为更多——更像你自己，更清醒地知道什么对你真正重要。围绕任务和截止日期设计的工具，到不了这里。

**Nation of One 从另一个前提出发：你是一个国家。**

国家不做优化。国家做治理——同时处理国防、外交、文化、经济、卫生、情报，每一个维度都不能缺席。衡量一个国家的标准，不是它产出了多少，而是它有没有在成为它本来可以成为的样子。

你是国家元首。AI 是你的幕僚长。八个部委，一种人生，完全属于你。

---

## 八个部委

| 代码 | 部委 | 管什么 |
|------|------|--------|
| NDRC | 国家发展和改革委员会 | 目标、OKR、重大决策 |
| MOF | 财政部 | 钱、时间、精力分配 |
| MOE | 教育部 | 学习、技能、有深度的输入 |
| NHC | 国家卫生健康委员会 | 睡眠、运动、饮食、身体信号 |
| MFA | 外交部 | 人脉、关系、社交能量 |
| MCT | 文化和旅游部 | 纯娱乐——休息就是休息，不用合理化 |
| MOD | 国防部 | 情绪韧性、风险管理、边界守护 |
| NIA | 国家情报局 | 行业前沿 + 兴趣圈追踪 |

## 核心功能

**普通建国**
回答几个问题，确定你的国家类型、发展阶段和各部委基线评级。当天就能开始运作。

**深度建国**
把你这几年的日记、笔记、聊天记录喂进去。系统用八个部委的视角读完你的数据，生成一份六维自我画像——优势区、盲点区、核心张力、能量地图、机会窗口、风险轨道。然后根据你真实的模式推荐该雇用哪些顾问，不是通用建议。

**每日提报**
快速过一遍八个部委。只记异常，正常的不用写。

**主动巡视**
每次对话开始前，系统自动扫描：几天没提报、哪个部委持续低分、顾问有没有新情报。不用你开口。

**内阁会议**
周报和月报，S/A/B/C/D 评级加趋势箭头。一次对话看清整个国家的健康状态。

**顾问团**
把真实的人——思想家、创业者、历史人物——蒸馏成顾问人格，任命到对应部委，随时召见参与你的决策。
[nuwa-skill](https://github.com/alchaincyf/nuwa-skill)（作者 [@alchaincyf](https://github.com/alchaincyf)）负责把人蒸馏出来，Nation of One 负责让他们上岗。

**情报简报**
用网络搜索实时追踪你的专业领域和兴趣方向。情报部真正在工作。

## 怎么用

| 你说 | 发生什么 |
|------|---------|
| `"建国"` | 普通建国流程 |
| `"深度建国"` / `"导入自我"` | rawdata → 自我画像 → 顾问推荐 |
| `"汇报"` | 每日部委提报 |
| `"开内阁会议"` | 周度评级和趋势分析 |
| `"月报"` | 月度宏观报告 |
| `"情报简报"` | 联网搜索你的关注领域 |
| `"定目标"` | 战略规划和 OKR |
| `"财政部怎么样"` | 单部委深度分析 |
| `"蒸馏 [人名]"` | 把一个人提炼成顾问人格 |
| `"任命 [人名] 为 [部委]"` | 将顾问任命到对应部委 |
| `"召见 [人名]"` | 召见顾问，以其视角回应当前问题 |
| `"顾问团状态"` | 查看所有顾问和任命情况 |

中文、英文、中英混合都可以，系统自动适配。

## 数据存储

所有数据存在 `~/Documents/nation/`，纯 Markdown，本地运行，不上云，不需要账号，全部属于你。

```
~/Documents/nation/
├── founding.md        # 建国宣言
├── config.md          # 部委配置 + 顾问任命
├── goals.md           # OKR（你准备好了再定）
├── logs/              # 每日提报
├── weekly/            # 内阁会议记录
├── monthly/           # 月度宏观报告
├── wiki/              # 从你的 rawdata 建立的部委知识库
└── advisors/          # 蒸馏好的顾问人格档案
```

## 设计原则

1. **先诊断，后规划** — 搞清楚你的国家现状，再谈目标
2. **休息就是休息** — 文旅部不等于教育部，娱乐不需要合理化
3. **数据极简** — 只记异常，正常不写
4. **主权在你** — 所有建议都是参考，元首说了算
5. **没有盲区** — 系统如实反映数据里看到的，包括你可能不想看到的
