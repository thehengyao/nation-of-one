# 深度建国流程

> 触发词："深度建国" / "导入自我" / "分析我的数据"
>
> 适用场景：用户有积累的个人数据（日记、笔记、聊天记录、内容创作等），想在建国前先做一次深度自我审计。
>
> 与普通建国的区别：普通建国靠问答，深度建国靠数据——先读你，再诊断，再建国。

---

## Phase 0：数据摄入

### 说明话术

"深度建国需要读你的原始数据。你可以：
- 直接把内容粘贴进来（日记、笔记、聊天记录、任意格式）
- 告诉我文件路径，我来读
- 分批提供，我每次处理完告诉你进度

数据越多越准，但哪怕只有一小部分也能开始。"

### 摄入规则

- 用户粘贴或指向数据 → 按 `{baseDir}/references/ministry-audit.md` 的提取模板逐部委提取信号
- 每批处理完立即写入 `~/Documents/nation/wiki/[部委代码].md`，不等全部完成
- 已处理的数据标记完成，下次不重复处理
- 告知进度："已处理 X 批数据，覆盖时间段 [X 到 Y]，继续提供或说'分析'进入下一阶段"

### 文件结构

```
~/Documents/nation/wiki/
├── NDRC.md    ← 目标、决策、重大转折点
├── MOF.md     ← 金钱、时间、精力模式
├── MOE.md     ← 学习路径、技能积累
├── NHC.md     ← 身体、睡眠、精力记录
├── MFA.md     ← 人际事件、关系模式
├── MOD.md     ← 情绪事件、压力源、应对方式
├── NIA.md     ← 信息输入、关注方向
└── MCT.md     ← 休息方式、快乐来源
```

## Phase 1：六维自我分析

### 触发

用户说"分析"或数据摄入完成后主动提出。

读取 `~/Documents/nation/wiki/` 所有文件，生成报告：

---

### 报告格式

```markdown
# [用户名] 国情分析报告
生成日期：YYYY-MM-DD
数据覆盖：[时间段]

## 优势区
你自然运作、不费力就做好的事。
不是你觉得自己擅长的，是数据里反复出现的。
[3-5条，每条附数据来源]

## 盲点区
rawdata 里反复出现，但你可能没有意识到的模式。
不是评判，是观察。
[3-5条，每条附具体例子]

## 核心张力
你身上真实存在的内在矛盾。
不是需要修复的缺陷，是你的复杂性。
[2-3组对立，描述两极]

## 能量地图
什么在给你充电，什么在漏电。
来自 NHC + MCT + MOD 的交叉信号。
充电：[列举]
漏电：[列举]

## 机会窗口
根据你的特质和当前外部环境，阻力最小的方向。
不是"你应该做什么"，是"你做什么会事半功倍"。
[2-3个方向]

## 风险轨道
如果不干预，根据历史模式你最可能掉进的坑。
[2-3个，每个附历史佐证]
```

---

### 报告呈现规则

- 报告呈现后，停下来等用户消化
- 说："这是数据给出的画像。你有什么想补充或不同意的？"
- 用户的补充意见和报告一起进入 Phase 2
- 存入 `~/Documents/nation/wiki/self-portrait.md`

## Phase 2：顾问推荐

### 逻辑

基于六维分析报告 + 用户补充意见，逐部委给出顾问推荐：

```
[部委]：根据你的 [具体模式/盲点]，你需要一个 [特质描述] 的人。
推荐：[人名]
理由：[一句话，紧扣你的具体情况，不是通用描述]
Token 成本：[高/中/低]（高 = 需要专门蒸馏对话）
```

### 推荐规则

- MCT 默认推荐"你自己"，不推荐外部人物
- 如果某部委用户已有明确想法，优先尊重用户判断
- 每个推荐附 token 成本提示
- 推荐完整列出后，问用户："哪几个你想先蒸馏？"

### 蒸馏决策

用户确认要蒸馏某人后：

1. 检查 `~/Documents/nation/advisors/` 是否已有现成档案
   - 有 → "已有 [人名] 的档案，直接任命还是重新蒸馏？"
   - 没有 → 提示使用 nuwa-skill 蒸馏，或手动提供内容

2. nuwa-skill 提示话术：
   > "[人名] 尚未蒸馏。建议使用 nuwa-skill 进行深度提炼（作者 @alchaincyf，`npx skills add alchaincyf/nuwa-skill`）。也可以直接粘贴此人的内容，我做基础提炼，精度较低。你选择哪种方式？"

3. 蒸馏完成后，档案存入 `~/Documents/nation/advisors/`，执行任命

## Phase 3：建国宣言更新

顾问团确认后，更新建国宣言：

读取（或创建）`~/Documents/nation/founding.md`，更新以下字段：

- 国家类型和发展阶段（基于六维分析）
- 各部门初始评级（基于 wiki 数据）
- 当前核心挑战（来自盲点区 + 风险轨道）
- 顾问团名单

存入 `~/Documents/nation/founding.md`，深度建国完成。

结束语："建国完成。你的国情画像已存档，顾问团就位。随时说'汇报'开始日常运作。"
