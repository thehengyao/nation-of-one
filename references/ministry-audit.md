# 部委审计提取模板

> 摄入 rawdata 时使用。每批数据按以下模板逐部委提取信号，写入对应 wiki 文件。
> 原则：只提取有信号的，没有就留空，不要捏造。

---

## NDRC 发改委
**看什么：** 目标、决策、重大转折、放弃了什么、坚持了什么

提取信号：
- 明确说过的目标或计划
- 重大决策及其理由
- 反复出现的方向摇摆
- 长期坚持的事（即使没说是"目标"）
- 放弃了但反复提及的事

写入格式：
```
## 决策记录
- [时间] [决策内容] → [结果/后续]

## 反复出现的方向
- [模式描述]

## 长期坚持
- [事项]
```

---

## MOF 财政部
**看什么：** 钱、时间分配、精力消耗

提取信号：
- 金钱焦虑的频率和触发点
- 时间实际花在哪里（不是计划，是实际）
- 精力高峰和低谷的规律
- 什么事情做完感觉"值"，什么感觉"亏"

写入格式：
```
## 金钱模式
- [焦虑触发点] / [消费模式]

## 时间实际分配
- [高频活动列表]

## 精力规律
- 高峰：[时间段/场景]
- 低谷：[时间段/场景]
- 感觉"值"的：[列举]
- 感觉"亏"的：[列举]
```

---

## MOE 教育部
**看什么：** 学了什么、停在哪里、输出了什么

提取信号：
- 主动学习的领域和方式
- 学了但没有输出/应用的（只吃不拉）
- 有深度输出的（写作、产品、教别人）
- 学习中断的时间点和原因

写入格式：
```
## 学习轨迹
- [领域] → [深度：浅尝/深入] → [有无输出]

## 停滞点
- [时间] [中断原因]

## 真正落地的
- [列举有实际输出的学习]
```

---

## NHC 卫健委
**看什么：** 睡眠、身体、精力、饮食信号

提取信号：
- 睡眠描述（时间、质量、异常）
- 身体抱怨（疼痛、疲劳、不适）
- 精力描述词（累、亢奋、虚、稳）
- 饮食提及
- 身体与情绪的连动（身体差时情绪如何）

写入格式：
```
## 睡眠模式
- [规律/不规律] [典型时长] [质量描述]

## 身体信号
- [高频出现的身体状态]

## 身体-情绪连动
- [观察到的规律]
```

---

## MFA 外交部
**看什么：** 人际事件、关系模式、社交感受

提取信号：
- 反复提及的人（正面/负面）
- 社交后的感受（充电/漏电）
- 人际冲突或紧张的触发点
- 主动维护的关系 vs 被动应付的关系
- 对"借光""消耗"等关系的描述

写入格式：
```
## 重要关系
- [人/群体] → [关系性质] → [给你的感受]

## 社交模式
- 充电型场景：[列举]
- 漏电型场景：[列举]

## 边界触发点
- [什么情况下你感到被侵犯/消耗]
```

---

## MOD 国防部
**看什么：** 情绪事件、压力源、应对方式、回避模式

提取信号：
- 高频情绪词（焦虑、愤怒、难过、错愕）
- 压力触发点
- 应对方式（做事、拖延、系统化、回避）
- 情绪被绑架的事件
- 韧性时刻（经历困难后的反应）

写入格式：
```
## 高频情绪
- [情绪] → [触发场景]

## 应对方式
- [典型应对模式，包括回避]

## 压力源
- [列举]

## 韧性时刻
- [经历困难后如何恢复的例子]
```

---

## NIA 情报局
**看什么：** 信息输入、关注方向、思维方式

提取信号：
- 反复关注的议题和领域
- 信息来源（主动搜索 vs 算法投喂）
- 思维方式（类比、辩证、第一性原理、叙事）
- 洞察和观点的质量（停在表面 vs 有穿透力）

写入格式：
```
## 持续关注的领域
- [领域列表]

## 思维特征
- [观察到的思维方式]

## 信息获取模式
- 主动：[渠道]
- 被动：[渠道]
```

---

## MCT 文旅部
**看什么：** 休息方式、快乐来源、放松场景

提取信号：
- 真正让你放松的事（不是"应该"放松的）
- 快乐的具体描述
- 娱乐被挤压的时间段
- 报复性放纵的记录（熬夜、暴食等）

写入格式：
```
## 真实充电方式
- [列举，越具体越好]

## 被挤压的时间段
- [记录]

## 报复性放纵模式
- [如有]
```

---

## 写入规则

1. 每批 rawdata 处理完，立即写入对应 wiki 文件，追加不覆盖
2. 用时间戳标注数据来源区间
3. 没有信号的维度留空，不猜测
4. 写完告知用户："已处理，[X] 个部委有新信号。"
