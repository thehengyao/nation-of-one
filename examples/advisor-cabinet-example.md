# 顾问团配置示例

这是一份根据个人特质定制的顾问团配置，供参考。
你的顾问团应该基于你自己的模式和需求来选择，不需要照搬。

---

## 示例背景

用户画像：设计师出身的独立产品人，跨学科背景（HCI + 艺术治疗），有想法但启动慢，吸收快但容易停在概念层，用系统回避情绪，对借光型社交高度警惕。

---

## 顾问团配置

| 部委 | 顾问 | 为什么选这个人 |
|------|------|--------------|
| NDRC 发改委 | 马斯克 + 拿破仑 | 马斯克拆解不可能，拿破仑不等完美条件就动。有想法但启动慢，需要这两种力 |
| MOF 财政部 | 洛克菲勒 | 历史上最系统性积累财富的人，心狠手稳，每一分钱有目的，不投机 |
| MOE 教育部 | 费曼 | 学到能教别人才算懂。吸收快但容易停在概念层，费曼逼你落地 |
| NHC 卫健委 | 乔布斯 | 直觉优先，活在当下，极简。容易用建系统回避身体信号，乔布斯逼你感受 |
| MFA 外交部 | 奥巴马 | 真实连接，不表演，长线关系。对借光型人高度警惕，需要一个懂真实建关系的模板 |
| MOD 国防部 | 荣格 | 处理痛苦的方式是建系统回避。荣格不让你绕开，整合阴影才是真正的韧性 |
| NIA 情报局 | 黑格尔 + 齐泽克 | 黑格尔找历史矛盾和趋势，齐泽克解剖意识形态。本来就用这套思维，装进来是本能 |
| MCT 文旅部 | 你自己 | 这个部委不需要顾问。你最懂怎么充电——蹦迪、看片、发呆，换个频道存在，是你自己的方式 |

---

## 使用方式

```
# 蒸馏顾问（需要 nuwa-skill）
npx skills add alchaincyf/nuwa-skill
# 然后对 Claude 说："蒸馏荣格"

# 任命
"任命荣格为 MOD"

# 召见
"召见荣格，我今天不想做一件必须做的事，他会怎么说"
```

---

## 说明

MCT 顾问填"你自己"不是占位符，是一个设计决定——
休息的方式是高度个人化的，没有人比你更懂你自己怎么充电。
顾问在这个部委里反而是干扰。
